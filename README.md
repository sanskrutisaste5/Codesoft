# CODSOFT Java Development Internship

## Task 1 - Number Game

A Java console application that generates a random number and allows the user to guess it with feedback such as Too High, Too Low, or Correct.

## Task 2 - Student Grade Calculator

Calculates total marks, average percentage, and assigns grades based on student performance.

## Task 3 - ATM Interface

A simple ATM system that allows users to deposit money, withdraw money, and check account balance.

## Task 5 - Student Management System

A console-based application that allows users to add, search, remove, and display student records using Java classes and ArrayList.
